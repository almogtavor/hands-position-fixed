# More Features!

### Speed

|Key/Commands|Function|
|:----------:|:-------|
|CapsLock+m| Move 5 words backward|
|CapsLock+.| Move 5 words forward|


### Case Changing

|Key/Commands|Function|
|:----------:|:-------|
|CapsLock+0|Upper Case|
|CapsLock+9|Lower Case|
|CapsLock+8|Title Case|
