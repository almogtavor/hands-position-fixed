# More Features!

### Speed ⚡️

|Key/Commands|Function|
|:----------:|:-------|
|CapsLock+m| Left X 6|
|CapsLock+,| Down X 5 |
|CapsLock+8| Up X 5 |
|CapsLock+.| Right X 6|


The intention of the speed bindings is to enable the user to move real fast:

### Case Changing

|Key/Commands|Function|

|CapsLock+9|Upper Case|
|CapsLock+0|Lower Case|
